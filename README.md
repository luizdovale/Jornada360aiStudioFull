## 📌 Visão Geral

O **Jornada360** é uma aplicação web moderna desenvolvida com **React + TypeScript + TailwindCSS**, criada para organizar, visualizar e acompanhar jornadas, atividades e conteúdos de forma simples, rápida e eficiente.

A proposta do projeto é entregar uma UI limpa, responsiva e otimizada, servindo como ferramenta funcional e como demonstração profissional das habilidades de desenvolvimento do autor.

## 🚀 Como Rodar o Projeto Localmente

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/luizdovale/Jornada360aiStudioFull.git
````

### 2️⃣ Acesse a pasta

```bash
cd Jornada360aiStudioFull
```

### 3️⃣ Instale as dependências

```bash
npm install
```

### 4️⃣ Configure variáveis de ambiente (se existirem)

Crie um arquivo `.env` ou `.env.local`:

```
VITE_API_KEY=sua_key_aqui
```

### 5️⃣ Inicie o servidor de desenvolvimento

```bash
npm run dev
```

Acesse o app em:
👉 **[http://localhost:5173](http://localhost:5173)**
